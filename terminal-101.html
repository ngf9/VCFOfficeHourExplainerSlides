The Basics: Three Terms to Know

Terminal

The app where you type commands. A text-based window into your computer's operating system.

Command Language

The syntax the terminal understands — Bash and Zsh are examples. The grammar you use to talk to your machine.

CLI

Command Line Interface — the text-based alternative to clicking buttons in a graphical interface (GUI).

What is Bash? Zsh? Oh My Zsh?

Bash

The original command language. Historically the default on macOS and Linux.

Zsh

A modern shell that replaced Bash as macOS default in 2019. Better defaults, smarter features.

Oh My Zsh

Not a shell — a framework on top of Zsh. Adds themes, plugins, and helpful shortcuts.

Why the Terminal Matters

Every tool you use — VS Code, Claude Code, npm, git — runs through the terminal under the hood. Learning a handful of commands gives you direct control over your development environment.

You only need about 10 commands to be productive.

Navigation: 3 Key Commands

$ pwd # Where am I? Shows your current folder
$ cd my-app # Go somewhere. Use cd .. to go back up
$ ls # What's in this folder? Lists files

Open Your Editor from Terminal

$ code .

Opens the current folder in VS Code. The fastest way to get your editor open with the right project loaded.

PRO TIP Make sure you've installed the code command from VS Code's command palette (Cmd+Shift+P → "Install code command")

npm: Two Commands You Need

$ npm install # Install all dependencies
$ npm run dev # Start the dev server → localhost:3000

You must run these inside your project folder. Check with ls — if you see package.json, you're in the right place.

When Things Go Wrong

command not found

You're in the wrong folder or the tool isn't installed.

EACCES permission denied

Try prefixing with sudo (carefully).

port already in use

Another dev server is running; kill it or use a different port.
